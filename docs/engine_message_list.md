# Engine Message List and What they use For?

* 1: static constexpr UINT _WM_ENGINE_TEST = WM_APP + 1; `(NOTE: WM_APP == 0x8000)`

* And as you can see `0x8001 is use for Engine Testing`!

* Why we need those custom message Check `check how_to_use_engine_message.md` for that!

* 2: static constexpr UINT _WM_ENGINE_EXIT = WM_APP + 2;

* Simple as name say `0x8002 is _WM_ENGINE_EXIT is tell engine to exit` simple!
