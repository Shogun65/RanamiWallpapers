# Ranami Wallpapers

Ranami Wallpapers is a lightweight live wallpaper application for Windows.

It lets you use videos as your desktop wallpaper with a simple interface for importing, managing, and switching wallpapers.

> **Status:** Beta

## Features

- 🎥 Live video wallpapers
- 📂 Import and save wallpapers
- 🖼️ Automatic thumbnail generation
- 📚 Wallpaper library
- 📋 Wallpaper information (resolution, duration, file size)
- ⚡ Double-click a wallpaper to apply it
- 🚀 Optional startup with Windows
- 💾 Automatically remembers your last wallpaper

## Installation

1. Download the latest release from the **Releases** page.
2. Run the MSI installer.
3. Launch **Ranami Wallpapers** from the Start Menu or Desktop shortcut.

## Requirements

- Windows 10 or Windows 11
